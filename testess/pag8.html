REGRA CSS

No CSS, temos a regra CSS, que se trata de um conjunto de instruções que segue uma sintaxe própria e que define como o estilo será aplicado a um ou mais elementos HTML.
Esta regra CSS, definidas pela W3C na sua forma mais rudimentar é composta de três partes diferentes:

Seletor {Propriedade: valor;}

Seletor: O elemento HTML para qual a regra será valida.
Propriedade: É o atributo do elemento HTML, sua caracteristica e comportamento.
Valor: É a especificação que define o aspecto da caracterisca, podendo ser definidos por diferentes tipos de dados.
