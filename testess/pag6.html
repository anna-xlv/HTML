VINCULAR O CÓDIGO CSS

CSS externo
Nesse método, o código CSS fica em um arquivo separado, geralmente com a extensão .css, que é vinculado ao HTML por meio da tag dentro do documento. 


<head> 
        <link rel="stylesheet" href="estilos.css"> 
</head>

O CSS externo é o mais usado para ligar CSS ao HTML pois o mantem em um documento separado deixando o HTML mais limpo e menor, e facilitar a definição da aparência do site.
