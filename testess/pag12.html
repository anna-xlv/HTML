SELETORES PRINCIPAIS

Seleção por classe: A seleção por classe por sua vez é feita atraves do nome da classe que foi definida como atributo no HTML, para essa forma de seleção o caractere ponto em seguida do nome da classe se faz necessario. 

<p class = "corAzul"> Texto simples 2 </p> 
        .corAzul{
            color: blue;
        }
