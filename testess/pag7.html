VINCULAR O CÓDIGO CSS

CSS interno
O CSS é inserido diretamente dentro do próprio arquivo HTML, por meio da tag style no bloco head. 


        <head> 
        <style>; 
        body {
            background-color: #f0f0f0;
        }
        </style>
        </head>

CSS inline
Os estilos são adicionados diretamente nos elementos HTML por meio do atributo style. 

< p style = "color: blue; font-size: 16px;"> 
        texto com css inline 
        < /p >
