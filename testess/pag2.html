O QUE É CSS?

- O que significa a sigla CSS?
A sigla CSS é para o termo em inglês Cascading Style Sheets, que se traduzido, significa folhas de estilo em cascata.


- Qual é a principal função dele?
A função do CSS é similar a de decorar uma casa, pense que o site é como um comodo da casa e cada comando do CSS é um objeto de decoração, nesse contexto você so vai arrumar o comodo da maneira que achar melhor, ou seja a função do CSS é de estilizar o site.
